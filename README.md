# Taake_learns_to_C
My journey learning the C language



### ***How to Run any project***
1. Clone repo:
   ```
   git clone https://github.com/HrafnSognir/Taake_learns_to_C.git
   ```
2. Go into the project folder:
   ```
   cd path/to/project
   ```
3. Compile the program using GCC:
   ```
   gcc <file>.c -o <app>
   ```
4. Run the program:
   ```
   ./<app>
   ```
