<details>
   <summary><strong><h3>How to Run any project:</h3></strong></summary>
   
  1. ##### Clone repo:
     ```
     git clone https://github.com/HrafnSognir/Taake_learns_to_C.git
     ```

  2. ##### Go into the project folder:
     ```
     cd path/to/project
     ```

  3. ##### Compile the program using GCC:
     ```
     gcc <file>.c -o <app>
     ```

  4. ##### Run the program:
     ```
     ./<app>
     ```
