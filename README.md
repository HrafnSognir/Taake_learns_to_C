  <summary><strong><span style="font-size: 1.1em;">How to Run any project:</span></strong></summary>
  <br>
  
  1. ##### Clone repo:
     ```
     git clone https://github.com/HrafnSognir/Taake_learns_to_C.git
     ```

  2. ##### Go into the project folder:
     ```
     cd path/to/project
     ```

  3. ##### Compile the program using GCC:
     ```
     gcc <file>.c -o <app>
     ```

  4. ##### Run the program:
     ```
     ./<app>
     ```
